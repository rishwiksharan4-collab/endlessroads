# Endless Roads
A lightweight, original cartoon 3D endless driving web game built with Three.js for mobile WebView apps.

Files:
- index.html — main website
- style.css — UI styles
- main.js — game logic (Three.js)
- privacy.html — privacy policy
- icon.png — app icon (replace with your PNG)
- feature_graphic.png — feature graphic (replace with final asset)
